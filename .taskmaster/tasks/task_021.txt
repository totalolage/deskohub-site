# Test Strategy:
## Test Strategy

1. **Functionality Testing**:
   - Verify the Training Room page loads correctly at `/training-room` and `/en/training-room` routes
   - Test that all components render properly with correct content
   - Confirm that images load correctly and are responsive
   - Test the "Book the Room" button links to the reservation page with the correct query parameter
   - Verify all external links (email, phone) have correct href attributes

2. **Internationalization Testing**:
   - Verify all text content is properly translated in both Czech and English
   - Test language switching between Czech and English versions of the page
   - Confirm that date formats, currency symbols, and other locale-specific content display correctly
   - Verify that SEO metadata is correctly localized

3. **Responsive Design Testing**:
   - Test the page layout on various screen sizes:
     - Mobile phones (320px - 480px)
     - Tablets (768px - 1024px)
     - Desktops (1024px+)
   - Verify that the grid layout properly adjusts from 1 column on mobile to 2 columns on desktop
   - Confirm that images are properly sized and maintain aspect ratio on all devices
   - Test that text remains readable and properly formatted at all screen sizes

4. **Performance Testing**:
   - Run Lighthouse performance tests to ensure the page loads efficiently
   - Verify that images are properly optimized and use the Next.js Image component correctly
   - Check that lazy loading is implemented for below-the-fold images
   - Test initial load time and ensure it's under 3 seconds on average connections

5. **Accessibility Testing**:
   - Run automated accessibility tests using tools like Axe or Lighthouse
   - Verify proper heading hierarchy (h1, h2, h3, etc.)
   - Confirm that all images have appropriate alt text
   - Test keyboard navigation throughout the page
   - Verify sufficient color contrast for all text elements
   - Ensure all interactive elements are accessible via keyboard and screen readers

6. **Cross-Browser Testing**:
   - Test the page on multiple browsers:
     - Chrome
     - Firefox
     - Safari
     - Edge
   - Verify consistent appearance and functionality across all browsers

7. **SEO Testing**:
   - Confirm that proper metadata is generated for both language versions
   - Verify Open Graph tags for social media sharing
   - Check that the page has a logical heading structure
   - Test that all images have descriptive alt text
   - Verify canonical URLs are correctly set for language versions
